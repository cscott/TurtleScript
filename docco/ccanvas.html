  • ¶

    Very simple canvas wrapper. This is intended to wrap a cairo canvas when running natively, or an HTML5 canvas when running in the browser. We export only the APIs we actually need (and don't require string parsing in the API) to make the native implementation easier.

    This isn't Simplified JavaScript -- we use try/finally here.

    BUGS FOUND: iOS -- the rect() primitive seems to improperly clear the area between 0,0 and the given x,y coordinate. I reimplemented in terms of the path primitives to work around. iOS - the text primitives don't seem to properly invalidate the rectangle the draw into. Therefore, unless there is some other drawing operation to expand the bounding box of the invalidation, the text will be clipped.

    C. Scott Ananian, May 13 2011
